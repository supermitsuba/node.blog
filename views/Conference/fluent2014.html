Security


  1. Client
  2. Message
  3. Server

Client Security


  1. Enable CSRF protection
  2. ESCAPE ALL THE CHARACTERS
  3. Have a CSP (Content-Security-Policy) header
  4. X-Frame-Option Headers
  5. Clean out bad headers
  6. XSS(Cross Side Scripting)
  7. HTTPS and Sessions should be Http Only

Message Level Security


  1. Use tokens, no session cookies
  2. Tokens can be stored in local storage
  3. Use httpOnly flag in Node

Server Security


  1. No Sql suffers from Injection
  2. Always check types
  3. Regular Expressions can cause DDOS
  4. use a try catch
  5. http://www.OWASP.org
  6. OWASP has tools!
